# Mafia: Nightfall Protocol - Roadmap

## 🎯 Vision
To create the most immersive, scalable, and secure digital social deduction experience, blending classic Mafia gameplay with modern cyber-noir aesthetics and advanced AI.

---

## 🛠 Phase 1: Foundation (Completed)
- [x] Monorepo structure (Vite, Node.js, Shared Logic).
- [x] Real-time game engine with phase management.
- [x] Persistent storage with Prisma & PostgreSQL.
- [x] Horizontal scaling with Redis & Socket.io Adapter.
- [x] Sophisticated Bot AI with personality and suspicion maps.
- [x] Production-grade security (PBKDF2, HMAC, Rate Limiting).
- [x] Dockerization & CI/CD pipeline.
- [x] Multi-platform support (Web, Mobile, Desktop integration).

## 🚀 Phase 2: Live Service Expansion (In Progress)
- [ ] **Advanced Anti-Cheat:** Behavioral analysis to detect "Teaming" patterns.
- [ ] **Mobile Parity:** Bring the web-client's cinematic UI to React Native.
- [ ] **Global Leaderboards:** Season-based rankings and "Nightfall Legend" tier.
- [ ] **State Recovery:** Redis-backed hydration to allow reconnecting to active games.

## 🎨 Phase 3: Player Expression & Economy
- [ ] **Store Integration:** Simulated bit-purchases for custom operative avatars and card skins.
- [ ] **Battle Pass System:** XP tracking and seasonal reward unlocks.
- [ ] **Clan/Guild Support:** Private channels and team-based tournaments.

## 🧠 Phase 4: Intelligence & Immersion
- [ ] **Voice Integration:** Proximity-based or phase-based spatial voice chat.
- [ ] **AI Game Master:** LLM-powered narrator to provide dynamic context and "flavor" to match resolutions.
- [ ] **Spectator Mode:** Cinematic "Eyes Only" view for high-rank matches.

---

## 📅 Maintenance & Security
- **Weekly:** Dependency audits and security patches.
- **Bi-Weekly:** Load testing for Redis/Socket scaling.
- **Monthly:** Behavioral model updates for Bot AI.
